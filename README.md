 -Marte Not Found — 404

**Houston, tenemos un problema... pero el viaje continúa.**  
Este proyecto forma parte del reto **“404s to Mars”**, donde el objetivo era diseñar y desarrollar una página de error 404 completamente responsive con una propuesta visual y narrativa única.

---

-Concepto

**Marte Not Found** es una interpretación moderna y humorística del clásico mensaje de error “404 Not Found”.  
Inspirado en la idea de una misión fallida hacia Marte, el proyecto combina estética retrofuturista, humor digital y referencias actuales a la inteligencia artificial.

> “Ni ChatGPT encontró vida en Marte.”

El concepto juega con la **ironía tecnológica**, mezclando la **búsqueda de vida inteligente** con la **dependencia de la inteligencia artificial**, dentro de una experiencia visual minimalista y experimental.

---

-Tecnologías Utilizadas

- **HTML5**
- **CSS3 (animaciones, gradientes, layout responsive)**
- **JavaScript (canvas para animaciones y minijuego)**
- **Canvas API** para el:
  - Fondo estelar animado.
  - Minijuego interactivo del astronauta.

---

-Descripción del Proyecto

El sitio presenta:
- Un **fondo estelar animado** simulando movimiento espacial.  
- Un **cartel central** con el mensaje “ERROR 404”.  
- Una frase meme que combina IA + humor digital.  
- Un **minijuego del astronauta**, donde el usuario puede saltar obstáculos y continuar la misión fallida.  

La experiencia fusiona **narrativa lúdica + estética computacional retro**, evocando pantallas de sistema antiguas pero con una ejecución moderna y fluida.
